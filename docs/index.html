Install

Drop the CSS bundle into your page. Add the JS bundle if you use any interactive form component (search clear, password toggle, stepper, switch, checkbox, radio, segmented, tabs, tag input, OTP, combobox, file, dropzone, character counter, rating, dialog).

<link rel="stylesheet" href="dist/brut.css">
<script src="dist/brut.js" defer></script>  <!-- only if you use a data-brut="…" component -->

All data-brut="…" components auto-init on DOMContentLoaded. For markup added later, call Brut.init(root) to wire any new instances. Re-init is a no-op.

Build

Source lives in src/tokens.css, src/components.css, and src/js/. Run the build to produce both bundles.

npm run build
# or
bash build.sh

# outputs:
#   dist/brut.css   ← tokens.css + components.css
#   dist/brut.js    ← src/js/core.js + src/js/components/*.js

Toast

Non-blocking transient notification. Programmatic API + auto-dismiss. A host element data-brut="toast-host" collects toasts; if none exists Brut.toast() creates one and appends it to <body>.

<!-- optional explicit host (corner placement) -->
<div class="brut-toast-host brut-toast-host--top-right" data-brut="toast-host"></div>

<script>
  Brut.toast({ kind: 'ok', message: 'Saved.', timeout: 4000 });
</script>

Tooltip

Flat ink bubble on hover/focus. data-brut="tooltip" on the trigger; tip text in data-brut-tip; optional side via data-brut-tip-side="top|bottom|left|right". Esc removes. The bubble is appended to <body> by JS and positioned with getBoundingClientRect().

<button class="brut-btn" type="button"
        data-brut="tooltip"
        data-brut-tip="Saved at 12:04"
        data-brut-tip-side="top">SAVE</button>

Menu

Context / overflow menu. Trigger has data-brut-menu-open="<id>"; the menu has matching id, data-brut="menu", and starts hidden. Click outside, press Esc, or click any item to close. Arrow keys move focus.

<button class="brut-btn" type="button"
        data-brut-menu-open="row-actions">ACTIONS</button>
<div class="brut-menu" id="row-actions" data-brut="menu" hidden>
  <button class="brut-menu__item" type="button">Edit</button>
  <button class="brut-menu__item" type="button">Duplicate</button>
  <hr class="brut-menu__sep"/>
  <button class="brut-menu__item brut-menu__item--danger" type="button">Delete</button>
</div>
