Why BRUT

Opinionated by design.

█

Hard borders only.

No soft shadows, no blur, no fake depth. Every edge is earned. Every shadow is a hard-offset black block. The brutalist constraint forces visual clarity.

{

Token-first system.

Every value — color, spacing, radius, motion — lives in a CSS variable. Override your brand in one block. No source maps, no build step, no rebasing.

♥

Zero lock-in.

Copy the CSS file. Done. No npm package, no framework requirement, no peer dependency hell. Drop it into any project — Rails, Django, plain HTML.

Install

Two files. Done.

No build step. No config. No npm install. Drop the two bundles into your HTML <head> and start composing with utility classes and data-brut hooks.

Download CSS Download JS 
<!-- index.html -->
<link rel="stylesheet" href="brut.css"/>
<script src="brut.js"></script>

</span>
