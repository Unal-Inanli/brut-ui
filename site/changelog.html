v0.1.0 Initial release Latest
May 3, 2026

The first public release of BRUT. Everything you need to build hard-edged, neo-brutalist interfaces — shipped as two flat files with zero dependencies.

New — Components

  • 150+ components across Forms, Interactive, Layout, Display, Typography, and Tables
  • Form controls: input, textarea, select, checkbox, radio, switch, stepper, range, range-dual, otp, tag-input, file, dropzone, rating, combobox, multiselect, password, date, time, color, search
  • Interactive: accordion, dialog, drawer, menu, popover, tabs, toast, tooltip, sidebar
  • Layout: stack, cluster, bar, grid, split, section, container, field, fieldset, topnav, footer
  • Display: badge, alert, card, tag, avatar, avatar-group, skeleton, spinner, empty, breadcrumbs, pagination, rows
  • Typography: full type scale from display-1 through caption, plus eyebrow, kicker, overline, code, pre, kbd, mark, quote, prose
  • Tables: table, table-wrap, table-toolbar, table-filter, table-columns with sticky head and sticky column support

New — Design System

  • 80+ CSS variables for color, typography, spacing, shadows, borders, and motion
  • Hard-edge shadow system: 6 sizes from --shadow-xs (2px) to --shadow-2xl (16px), never blurred
  • 6 pop colors: pink, lime, blue, orange, purple, mint — plus semantic signal colors
  • Snap motion tokens: --dur-fast 80ms, --dur-base 140ms, --ease-snap cubic-bezier
  • Three font families: Archivo Black (display), Space Grotesk (UI), JetBrains Mono (code)
  • 4px base spacing grid from --sp-1 (4px) to --sp-20 (80px)

New — JS Runtime

  • 32 interactive components auto-initialized via data-brut attribute hooks
  • All components dispatch brut:change and brut:complete custom events
  • All interactive components mirror state to a hidden <input> for form compatibility
  • Full keyboard support: Space/Enter to activate, Arrow keys for navigation, Escape to close
  • Idempotent initialization — safe to call multiple times, works with dynamic DOM

New — Site & Docs

  • Marketing site (site/) with landing, component gallery, examples, get started, and changelog pages
  • Full documentation (docs/index.html) with live component previews and code snippets
  • 52 standalone preview pages for isolated component testing
  • 2 full-page demo compositions: landing page and login form
