Quick start ~30 seconds

Get started.

BRUT ships as two flat files. No npm, no build step, no config. Drop them in, compose with classes.

01

Add the files

Download both files from the latest release and add them to your <head>. Or link directly to the CDN.

<!-- In your <head> --> <link rel="stylesheet" href="brut.css" /> <script src="brut.js" defer></script>
i
JS is optional. Interactive components (switch, tabs, dialog, etc.) need brut.js. Pure CSS components work without it.
Download brut.css Download brut.js
02

Your first component

Every BRUT component is an HTML element with a .brut-* class. Interactive components also get a data-brut attribute so the JS runtime can wire them up.

A button

<button type="button" class="brut-btn brut-btn--primary"> Get loud </button>

Result

A toggle switch (interactive)

<label class="brut-cluster brut-cluster--sm" style="cursor:pointer"> <span class="brut-switch" data-brut="switch"> <input type="checkbox" hidden /> <span class="brut-switch__knob"></span> </span> Enable notifications </label>

Result

A form field

<div class="brut-field"> <label class="brut-field__label" for="email">Email</label> <input id="email" class="brut-input" type="email"/> <span class="brut-field__hint">We never share your email.</span> </div>

Result

We never share your email.
03

Customize tokens

Every value in BRUT is a CSS variable. Override the ones you want in your own stylesheet — no recompiling, no rebasing, no source maps.

/* your-theme.css — override after brut.css */ :root { --primary: #FF6B9D; /* swap yellow → pink */ --font-display: 'Arial Black', sans-serif; --shadow-md: 6px 6px 0 0 #FF6B9D; /* colored shadows */ }

Default (yellow)

Overridden (pink)

✓
That's it. One :root block changes your entire UI instantly — colors, spacing, shadows, motion.
